## Запуск проекта
+ Скачать проект:
  + `git clone https://github.com/a-shdv/products-warehouse.git -b develop`
+ Перейти в папку с проектом:
  + `cd products-warehouse`
+ Находясь в директории проекта, запустить следующую команду:  
  + `docker compose up -d`

## Swagger
